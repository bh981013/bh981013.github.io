# 나의 첫번째 게시물
첫번째 게시글은 앞으로의 원활한 게시글 작성을 위해 마크다운 문법을 알아보도록 하자.   
<br/>

1.헤더
-------
* 큰헤더 
    ```
    큰 헤더를 작성해보자
    ================
    ```
    <br/>
* 작은헤더
    ```
    작은 헤더를 작성해보자
    ----------------------------
    ```
    <br/>
* #헤더
    ```
    # h1
    ## h2
    ### h3
    ```

## 3. 강조

    ```
    __볼드__ or *볼드*
    ```
