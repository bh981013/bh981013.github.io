# 선택자

## 기본 선택자

- \*    : 모든 요소를 선택

- _elmementName_    : elmentName 이라는 이름을 가진 모든 요소를 선택

- .className : className 이라는 class 속성을 가진 요소를 선택

- \#idName  : idName이라는 id 속성을 가진 요소를 선택
